CSS, JS: los tres pilares del desarrollo web.

HTML, CSS y JS son los tres lenguajes fundamentales para el desarrollo web. HTML se utiliza para estructurar el contenido de una página web, CSS darle estilo y JS para añadir funcionalidad.

HTML

HTML significa HyperText Markup Language. Es un lenguaje de marcado que se utiliza para crear el esqueleto de una página web. HTML se compone de etiquetas que indican al navegador web cómo mostrar el contenido de la página. Algunas de las etiquetas HTML más comunes son:

  • h1: encabezado de nivel 1
  • p: párrafo
  • img: imagen
  • ul: lista no numerada
  • ol: lista numerada

CSS

CSS significa Cascading Style Sheets. Es un lenguaje de hojas de estilo que se utiliza para dar estilo al contenido de una página web. CSS se puede utilizar para cambiar el color, el tamaño, la fuente y la alineación del texto, así como el tamaño, la posición y el comportamiento de las imágenes, los vídeos y otros elementos de la página. Algunas de las propiedades CSS más comunes son:

  • color: el color del texto
  • font-size: el tamaño del texto
  • font-family: la fuente del texto
  • text-align: la alineación del texto
  • width: el ancho de un elemento
  • height: la altura de un elemento

JS

JS significa JavaScript. Es un lenguaje de programación que se utiliza para añadir funcionalidad a una página web. JS se puede utilizar para realizar tareas como:

  • Validar formularios
  • Animar elementos de la página
  • Crear juegos y aplicaciones web

Algunas de las funciones JS más comunes son:

  • alert(): muestra un mensaje de alerta al usuario
  • querySelector(): selecciona un elemento de la página web
  • addEventListener(): añade un evento a un elemento de la página web
  • console.log(): imprime un mensaje en la consola del navegador

Conclusión

HTML, CSS y JS son los tres lenguajes fundamentales para el desarrollo web. Cada uno de ellos tiene su propio propósito y se complementan entre sí para crear páginas web completas y funcionales.
